# J.Oli Origins Demo Assets

Tyto soubory jsou aktuálně použité v demo homepage.

## Jak nahradit za plnou kvalitu
1. Zachovejte stejné názvy souborů.
2. Přepište existující soubory novými verzemi (vyšší rozlišení).
3. Není potřeba měnit kód.

## Mapování souborů
- `logo.png` -> logo v headeru a footeru
- `complete-set.jpg` -> blok kompletní sady
- `no1-cleanser.jpg` -> produkt No1 + hero
- `no2-toner.jpg` -> produkt No2 + hero
- `no3-collagel.jpg` -> produkt No3 + hero
- `no4-day-serum.jpg` -> produkt No4 denní + hero
- `no4-night-serum.jpg` -> produkt No4 noční + hero
